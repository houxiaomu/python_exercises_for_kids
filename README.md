# Python Programming Exercises

Welcome to your Python practice exercises! These exercises will help you learn programming step by step.

## How to Use

1. Open any exercise file in PyCharm
2. Read the instructions at the top of the file
3. Write your code where it says `# YOUR CODE HERE`
4. Run the file (Right-click -> Run, or press Ctrl+Shift+F10 / Cmd+Shift+R)
5. Check if your tests pass!

## Recommended Learning Path

1. **Start with List Exercises** - Learn basic loops and lists
2. **Try Math Exercises** - Practice calculations
3. **Do Logic Exercises** - Master if/else conditions
4. **Have fun with Pattern Exercises** - Visual programming!
5. **Play with Game Exercises** - Apply what you learned
6. **Challenge yourself with Array & String** - Advanced topics
