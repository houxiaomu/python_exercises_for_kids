# Python Programming Exercises

Welcome to your Python practice exercises! These exercises will help you learn about arrays (2D lists) and string operations.

## How to Use

1. Open any exercise file in PyCharm
2. Read the instructions at the top of the file
3. Write your code where it says `# YOUR CODE HERE`
4. Run the file (Right-click -> Run, or press Ctrl+Shift+F10 / Cmd+Shift+R)
5. Check if your tests pass!
